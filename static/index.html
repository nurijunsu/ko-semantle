게임 방법

코맨틀은 단어의 의미적 유사도를 이용해 숨겨진 정답 단어를 맞히는 게임입니다.

규칙

  • 매일 새로운 정답 단어가 출제됩니다.
  • 아무 한국어 단어를 입력하면 정답과의 유사도와 순위를 알려줍니다.
  • 유사도는 두 단어의 의미적 거리를 나타냅니다. (높을수록 가까움)
  • 순위는 사전 내 모든 단어 중 정답에 가까운 순서입니다.
  • 유사도와 순위를 단서로 정답을 추리해보세요!

색상 안내

상위 10위 상위 100위 상위 1,000위 1,000위 밖

팁

  • 처음에는 다양한 주제의 단어를 넓게 시도하세요.
  • 순위가 1,000위 안에 들면 정답에 가까워지고 있다는 뜻입니다.
  • 유사도 점수 차이를 비교하며 방향을 잡으세요.

🎉 정답입니다!

클립보드에 복사되었습니다!
